#  AtmaCinema - Movie Booking App


AtmaCinema is a modern mobile application for booking movie tickets with a seamless seat selection experience. Built using **Flutter** and **Laravel**, it allows users to browse movies, select seats, place orders, and download tickets in **PDF format**.


##  Tech Stack

- **Frontend**: Flutter (Dart)
- **Backend**: Laravel (REST API)
- **Database**: MySQL 



##  Key Features

-  Real-time seat selection and booking system
-  Complete order & checkout flow
-  Ticket history with booking details
-  Downloadable & printable PDF tickets
-  Admin dashboard for managing movies, showtimes, and seats (optional)
-  User authentication
-  Responsive UI for Android devices
